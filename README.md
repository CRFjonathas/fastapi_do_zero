# FastAPI do Zero 🚀

Repositório de acompanhamento do curso [FastAPI do Zero](https://fastapidozero.dunossauro.com/) ministrado por Eduardo Mendes (@dunossauro). O objetivo principal é construir uma API de gerenciamento de tarefas completa passo a passo.

## 🛠️ Tecnologias e Ferramentas
- **FastAPI & Pydantic**: Criação da API e validação de dados.
- **SQLAlchemy & Alembic**: Modelagem do banco de dados e migrações.
- **pytest**: Testes automatizados.
- **Poetry**: Gerenciamento de dependências e ambiente virtual.
- **Docker & Fly.io**: Conteinerização e deploy em produção.
